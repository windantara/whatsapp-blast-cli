# Report Directory

This folder contains blast reports generated by WhatsApp Blast CLI.

## Report Format

Each report is named with the following format:
```
[timestamp]-[message-file]-[number-file].txt
```

Example:
```
1705320150-welcome-message-customer-list.txt
```

## Report Content

Reports contain:
1. List of all numbers with their status (SUCCESS/FAILED)
2. Name associated with each number (if provided)
3. Error message for failed sends
4. Statistics summary at the end

## Example Report Content

```
6281234567890 | SUCCESS | John Doe
6281234567891 | SUCCESS | Jane Smith
6281234567892 | FAILED | Bob Johnson | Number not registered on WhatsApp

==================================================
BLAST STATISTICS
==================================================
Total: 3
Success: 2
Failed: 1
Success Rate: 66.67%
```

## Tips

- Review reports after each blast
- Use reports for follow-up on failed messages
- Keep reports for audit trail
- Archive old reports periodically
